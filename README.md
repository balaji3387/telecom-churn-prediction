# Telecom Customer Churn Prediction

Predicting customer churn using XGBoost, SHAP, and SMOTE.

## Tools Used
- Python, scikit-learn, XGBoost, SHAP, imbalanced-learn

## Results
- AUC: 0.91 | F1: 0.83 | Recall: 81%

## Key Highlights
- Engineered 8 new features from raw telecom data
- Fixed class imbalance using SMOTETomek
- Used SHAP to identify top 3 churn drivers
